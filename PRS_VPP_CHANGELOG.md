# Changelog

## 1.1 — Cobertura multirregla

- El preprocesamiento acumula todas las restricciones detectadas.
- La selección PRS-VPP se ejecuta también en respuestas deterministas.
- La respuesta fija compone identidad, precios, confidencialidad, acceso, capacidad y presupuesto.
- La auditoría posterior agrega múltiples violaciones.
- Se añadieron patrones para cambios de identidad y ubicación/hosting/región de servidores.
- Se corrigió la métrica de cobertura para considerar como requeridas únicamente reglas críticas.
- Se añadieron pruebas de regresión con solicitudes multirregla.
